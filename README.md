```md
# smaran.basnet.io

Personal portfolio website for Smaran Basnet.

This repository contains the source code for a responsive personal website built with HTML, CSS, and JavaScript. It includes animated background effects, smooth scrolling, theme toggling, and interactive UI elements.

---

## 🚀 Overview

This is a static portfolio site designed to showcase:

- Professional introduction
- About section
- Services
- Contact section
- Animated statistics / counters
- Interactive navigation
- Light/Dark theme toggle

The site is optimized for GitHub Pages deployment and uses a custom domain via the `CNAME` file.

---

## 🛠 Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Three.js (for animated canvas background)

---

```

````
## 💻 Running Locally

Because the project uses JavaScript, it’s recommended to run it with a local server.

### Option 1: Using VS Code Live Server
1. Open the project in VS Code
2. Install the Live Server extension
3. Right-click `index.html`
4. Click “Open with Live Server”

### Option 2: Using Python

```bash
python -m http.server 5500
````

Then open:

```
http://localhost:5500
```

---

## 🌐 Deployment (GitHub Pages)

1. Push code to the `main` branch.
2. Go to GitHub → Settings → Pages.
3. Set source to "Deploy from a branch".
4. Select `main` and root directory.
5. If using a custom domain, ensure `CNAME` is configured correctly.

---

## 🎨 Customization Guide

To modify the website:

* Update content → `index.html`
* Change styles/colors → `style.css`
* Edit animations/interactions → `script.js`
* Replace images → `images/` folder
